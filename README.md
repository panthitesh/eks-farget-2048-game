# 2048 Game on AWS EKS using Fargate
This project demonstrates an end-to-end deployment of the 2048 game
on AWS EKS using Fargate and Application Load Balancer (ALB).
## Technologies Used
- AWS EKS
- AWS Fargate
- Kubernetes
- ALB Ingress Controller
## Outcome
A public 2048 game hosted on AWS without managing EC2 servers.
